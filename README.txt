爬取新闻资讯的爬虫程序，所含文件说明如下：

--------------------mySpider.py---------------------
mySpider.py是主程序，运行mySpider.py，根据提示输入想要搜索的关键词，即可获得相应的新闻和链接
mySpider.py中line 122指循环次数，即，改变range()中的数字可以指定想要爬取的页数

--------------------mytest.py------------------------
mytest.py是测试程序，运行mytest.py，可以知道语句的分割形式，是否会被写入csv，若写入，可以知道它的行业类型分类
改变line 16中title=''中字符串的值，可以测试指定语句

--------------------areas.txt-------------------------
areas.txt中是地区列表
在此列表中的地名将在“涉及方类型”中被归类为“地区”

--------------------parties.txt-----------------------
parties.txt中是组织列表
在此列表中的组织名称将在“涉及方类型”中被归类为指定组织

--------------------fields.txt------------------------
fields.txt中是细分行业列表
在此列表中的细分行业名称将在“涉及方类型”中被归类为指定行业类型

--------------------stopwords.txt----------------------
stopwords.txt中是停止词列表
包含停止词且行业类型为“待定”的语句将不会被写入csv文件中

--------------------userdict.txt-----------------------
userdict.txt中是自定义分词列表
一些专业词汇、特定词汇无法被原有jieba中正确分割的，可以在这里指定
后面数字代表频次，只要足够大，具体数值不重要


--------------------results.csv-----------------------
results.csv中是运行mySpider.py后输出的爬取结果，是utf-8编码格式
如果要在excel工作表中打开，需要先选择打开方式-记事本，再点击文件-另存为-编码方式选择ANSI，即可在excel中以正常格式打开
